# 曜日別対応の概要

曜日ごとの入稿・掲載・問い合わせが偏るため、各曜日に担当タスクを割り振っています。基本原則は次の通りです。

1. **月曜：仕込みと確認** — 週全体の段取りを固める。
2. **火曜：原稿整理** — 取材メモを原稿化し、差し込み枠を埋める。
3. **水曜：中日のフォロー** — 進捗遅延を洗い出し、関係者へリマインド。
4. **木曜：校了準備** — 原稿を確定させ、デザインと進行を束ねる。
5. **金曜：締切対応** — 週内リリースをまとめ、週末に備える。
6. **土曜：休日運用** — 最低限の監視と問い合わせ回答。
7. **日曜：週次レポート** — KPIの速報を作成し、月曜朝会で使える状態にする。
