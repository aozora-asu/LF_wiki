# 土曜日の対応 (休日監視)

1. **CMSの予約確認**  
   - 午前中に当日分の公開予約が崩れていないかチェック。  
   - 問題があれば即座に公開し、チャットで報告。

2. **問い合わせ一次対応**  
   - 共有アドレス宛のメールを2時間おきに確認し、テンプレ回答で返信。  
   - 即答できない場合は「月曜10:00までに回答」と約束し、Notion に記録。

3. **SNSモニタリング**  
   - 公式ハッシュタグの言及を Hootsuite で監視し、異常があればスクショを残す。

4. **トラブル連絡フロー**  
   - 重大インシデントは `incident@company` と電話連絡網で共有。  
   - 連絡時は「事象 / 影響範囲 / 初動内容」を3行でまとめる。
