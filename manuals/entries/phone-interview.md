# 電話取材のやり方

電話取材は「主旨説明 → 同意確認 → ヒアリング → まとめ」の4ステップで進めます。以下のチェックリストを順番に確認してください。

## 1. 事前準備

- 前回の掲載記事と出稿簿を確認し、掲載目的を把握する。
- 5W1Hの質問シートを印刷し、聞き漏れがないようにする。
- 録音機器（iPhone または IC レコーダー）をテストし、相手に録音許可をとる。

## 2. 初動スクリプト

```
「いつもお世話になっております。〇〇編集部の△△です。
  本日は□□の記事について5分ほどお時間よろしいでしょうか？」
```

- 同意が得られたら、録音の可否と掲載媒体名を改めて伝える。
- 取材対象が忙しい場合は再度の通話時間を設定し、Teams カレンダーに記録する。

## 3. ヒアリングのコツ

1. ファクト確認（会社名・担当部署・肩書）
2. 目的と背景（なぜ今回の施策を行うのか）
3. 詳細（数値・固有名詞・時系列）
4. コメント（読者へのメッセージ）

数字や固有名詞は復唱して確認します。特に金額・日付・人数の3点は必ず再確認すること。

## 4. まとめとフォロー

- ヒアリング内容を 10 分以内に要約し、Teams の `#interview-log` にノートを共有する。
- 掲載予定日と校了スケジュールを再度伝え、メールでサマリーを送付する。
