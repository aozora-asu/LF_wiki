# 出稿簿の書き方

出稿簿は「誰が・いつ・どこに・いくらで」掲載するかを一目で確認するための帳票です。共有PCでは `共有ドライブ > 広報 > 出稿簿.xlsx` を更新します。

## 基本ルール

1. **案件名**: 媒体名＋企画名を30文字以内にまとめます。
2. **入稿予定日**: 西暦表記で `2025/04/01` の形式に統一します。
3. **金額**: 税抜きの媒体費用のみを記載し、備考に手数料などを記入します。
4. **ステータス**: `検討中 / 進行中 / 入稿済 / 掲載済` のいずれかを選びます。

## 記入手順

1. **最新行をコピー**: 直近の最終行を選択し、1行下に複製してフォーマットを揃えます。
2. **案件情報を更新**: 案件名・広告主・担当者・金額を入力します。
3. **入稿スケジュールを反映**: 入稿予定日と媒体の締切時間を入力し、備考に素材の受け渡し方法を記載します。
4. **共有メモ**: 変更があった場合は Teams の `#ad-weekly` チャネルに「出稿簿更新」のテンプレを流します。

## よくあるミスと対策

- **日付のズレ**: 入稿日と掲載開始日を取り違えないように、両方を備考に明記する。
- **担当欄の空欄**: バイト担当・社員サポートの2名を書く。緊急連絡先も備考に追記。
- **複数案件の混在**: 媒体が同じでも案件ごとに1行ずつ作成し、色を付けてグループ化する。

## 参考フォーマット

```
案件名 / 広告主 / 担当 / 入稿予定日 / 掲載開始 / 金額(税別) / ステータス / 備考
```

上の順序を崩さず入力すれば、週次報告書にそのまま貼り付けできます。必要なテンプレートは `manuals/media/templates/shukkobo.xlsx` に格納されています。
