# 金曜日の対応 (週末締切)

1. **最終出稿確認**  
   - 当日掲載分を `公開済` へ変更し、リンクを出稿簿に貼る。  
   - 未掲載があれば原因をコメント欄に残す。

2. **週末体制の引き継ぎ**  
   - 土日担当者へ「公開予定 / 注意事項 / 緊急連絡先」をまとめて共有。  
   - 共有テンプレは `manuals/media/templates/weekend_handover.md`。

3. **請求関連の下準備**  
   - 月内完了分の媒体費を Excel に転記し、翌週請求書を即出せるようにする。

4. **週次ふりかえり**  
   - KPI ダッシュボードを更新し、良かった点・改善点をそれぞれ3つ書き出す。  
   - まとめは Slack の `#weekly-review` にアップ。
