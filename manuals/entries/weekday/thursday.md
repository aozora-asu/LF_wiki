# 木曜日の対応 (校了準備)

1. **原稿最終確認**  
   - 文末の表記ゆれ（「です・ます」調）を全件チェック。  
   - 社名・肩書は最新表記を広報リストと突合。

2. **画像確認**  
   - 写真のクレジット、トリミング可否を担当者に確認。  
   - 解像度不足のものは即座に差し替え依頼を送る。

3. **デザイン連携**  
   - Figma のコメントをすべて Resolve し、残作業を ToDo リスト化。  
   - 18:00 までに進行管理表へ「校了予定時刻」を記載。

4. **週末公開記事の予約**  
   - CMS で公開予約時刻を設定し、プレビューURLを共有。  
   - 予約完了後は Slack の `#release-notice` に記録。
